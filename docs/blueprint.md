# **App Name**: CyberTrace Toolkit

## Core Features:

- Tool Selection Page: A landing page that lists all OSINT tools with descriptions and links to individual tool pages.
- Username Lookup: Enter a username and retrieve associated information (data will be based on the provided backend code).
- Phone Number Validation: Enter a phone number and validate its format and carrier information (data will be based on the provided backend code).
- Reverse Image Search: Upload an image and find visually similar images on the web (data will be based on the provided backend code).
- IP Locator: Enter an IP address and get its geographical location and associated information (data will be based on the provided backend code).
- Domain Lookup: Enter a domain name and retrieve information about its registration, DNS records, and hosting (data will be based on the provided backend code).
- Email Lookup: Enter an email address and retrieve associated information (data will be based on the provided backend code).

## Style Guidelines:

- Primary color: Electric Green (#7CFC00) for a high-tech, cyber feel.
- Background color: Dark navy blue (#1A232E) for a serious, professional, almost futuristic look.
- Accent color: Cyan (#00FFFF) to highlight interactive elements.
- Body and headline font: 'Space Grotesk' sans-serif for a techy and scientific feel, use 'Inter' for the body text.
- Use minimalist, neon-style icons for each tool category.
- Use any preferable layout for the tool selection page.
- Subtle glow effects on tool cards upon hover.
